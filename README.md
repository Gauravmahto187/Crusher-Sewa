## Crusher Material Sewa

Monorepo for a rental/logistics platform. Contains a Node/Express/MongoDB API (`backend`) and a Vite + React + Tailwind frontend (`frontend`).

### Backend
- Tech: Node.js, Express, MongoDB (Mongoose).
- Entry: `src/server.js`.
- Dev server: `npm run dev` (uses nodemon).

### Frontend
- Tech: React 19, Vite, Tailwind CSS, React Router.
- Entry: `src/main.jsx`, `src/App.jsx`.
- Dev server: `npm run dev` (Vite).

### Getting started
1. Install deps
   - `cd backend && npm install`
   - `cd frontend && npm install`
2. Environment
   - Create `backend/.env` with `MONGO_URI=<connection string>` and optional `PORT`.
3. Run dev servers
   - Backend: `npm run dev` from `backend/`.
   - Frontend: `npm run dev` from `frontend/` (see the printed local URL).
